# Emotion Detection Learning Support Engine

## Overview

Emotion Detection Learning Support Engine is an AI-powered educational application that detects a user's emotion from text and generates personalized learning support using Google Gemini AI.

The application provides:

- Emotion Detection
- AI-generated Learning Suggestions
- PowerPoint Presentation Generation
- Image Generation Support
- OCR Text Extraction
- Streamlit Web Interface

---

## Features

- Detect emotions from user text
- Personalized learning recommendations
- Generate presentation content
- Upload text or image files
- Clean and simple user interface

---

## Technologies Used

- Python
- Streamlit
- TensorFlow
- BERT
- BiLSTM
- Google Gemini API
- Transformers
- OpenCV
- Python-PPTX

---

## Installation

Create a virtual environment:

```bash
python -m venv venv
```

Activate it:

Windows:

```bash
venv\Scripts\activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
python -m streamlit run app.py```
